== Description ==

This countdown plugins allows you to insert a simple countdown as a widget on your blog - Counting up to any event!

You simply need to alter a few settings in the admin panel to configure the event and enable the widget - The countdown will work automatically!

== Installation ==

1. Upload `jr-countdown.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Countdown Settings in your Administration Panel
4. Enable the widget through the Appearance --> Widgets tab.
